Git博客搭建

前言

本博客环境都是在win下搭建完成

nodejs版本更新

后面用到的主题需要最新版本支持，旧版本无法编译通过，因此到官网重新下载最新包覆盖安装。

搭建github博客

新建仓库

git上新建一个仓库，名字必须是username.github.io , 其中username是你的用户名

配置ssh Key

提交代码到github需要权限才可以。
./: ssh 检测是否有安装ssh,没有就安装git，自带ssh安装。
ssh-keygen -t rsa -C “git上设置的邮件地址”
连续回车3次生成.pub文件。拷贝里面的全部到github主页，
进入安全设置，SSH and GPG keys -> New SSH key

测试

ssh -T git@github.com

设置git用户名和密码

./:  git config --global user.name "koliy" //用户名
./:  git config --global user.email "xxxx@126.com" //git上注册的邮箱

安装hexo

npm install -g hexo

创建hexo博客目录

创建一个hexo目录

cd hexo
heox init

生成静态文件

hexo g

预览

hexo s

下载博客主题

官网主题
我下的是icarus.
然后放到themes目录下
修改_config.yml指定对应的主题
theme: icarus
从新生成，启动就可以看到了

上传到git

修改_config文件

deploy:
type: git
repository: https://github.com/koliy/koliy.github.io
branch: master

执行上传指令

hexo d -g
